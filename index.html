Medication Reminder Notification System

Developed a Python-based desktop application that utilizes the plyer library to create a simple yet effective medication reminder system. The application generates daily notifications to remind users to take their medication on time. Key features include:

  • Notifications: Configured to send a notification with a custom title and message at a specified interval.
  • User-Friendly Design: Integrated an icon and set a notification timeout for enhanced user experience.
  • Efficiency: Implemented a loop to run the program continuously, ensuring timely reminders every day.
    Tools/Technologies: Python, Plyer, Time module
